Drawing

Drawing is a visual art that uses an instrument to mark paper or another two-dimensional surface.
The instruments used to make a drawing are pencils,crayons,pens with inks,brushes with paints,
or combinations of these, and in more modern times,computer styluses with graphics tablets or gamepads in VR drawing software.



History:

Drawing is one of the oldest forms of human expression,with evidence for its existence preceding that of written communication

css

(Cascading Style Sheets) is a simply designed language intended to simplify the process of making web pages presentable. CSS allows you to apply styles to HTML documents. It describes how a webpage should look. It prescribes colors, fonts, spacing, etc. In short, you can make your website look however you want. CSS lets developers and designers define how it behaves, including how elements are positioned in the browser.
